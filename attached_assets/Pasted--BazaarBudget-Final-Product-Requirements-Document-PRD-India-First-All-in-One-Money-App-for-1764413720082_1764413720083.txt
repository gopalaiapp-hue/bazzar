### Complete Feature List – Nothing Missing (Grouped by User Journey)

#### 1. Onboarding (3 minutes max)
- Phone number + OTP  
- Auto-read last 12 months SMS & WhatsApp payments (permission once)  
- Detect salary credits → auto-create “Salary” pocket  
- Detect rent debit → auto-create “Rent” pocket  
- Ask 3 questions only:  
  → Married? Yes/No  
  → Parents staying with you?  
  → Do you have freelance / shop / agriculture income?  

#### 2. Money Pockets System (The Killer Feature)
Every rupee lives in a pocket. User sees 5–8 pockets on home screen:

| Pocket Type       | Auto-created? | Examples                              | Money Movement Rules                              |
|-------------------|-------------------------------------------------------|----------------------------------------------------|
| Cash              | Yes (camera scan or manual)                           | Only outflow, never inflow                         |
| Bank – SBI        | Auto from SMS                                         | Full visibility + transactions                     |
| Bank – HDFC       | Auto                                                  |                                                    |
| UPI Wallet        | Auto (PhonePe, GPay, Paytm)                           |                                                    |
| Salary            | Auto (highest monthly credit)                         | 50–70% auto-routed to bills & savings              |
| Freelance / Shop  | Auto or manual                                        | Irregular – AI predicts next date                  |
| Rental Income     | Auto (if regular credit + user confirms)              |                                                    |
| Agriculture       | Manual once a year                                            | Exempt from tax calculation                        |
| Family Vault      | Created on invite                                             | Shared between husband-wife or parents             |

User can drag-drop money between pockets in 2 taps.

#### 3. Expense Tracking – Zero Effort
- 95% auto-categorised (Zomato → Food, Swiggy → Food, Amazon → Shopping)  
- Cash expense: One-tap voice note “₹320 petrol” or photo of bill → OCR + AI  
- Split expense: Long-press any transaction → “Split with wife 60:40” → auto UPI request  
- Daily 8 PM brief notification (user sets time):  
  Example → “Aaj ka kharcha ₹1,840 | Food ₹920 | Petrol ₹320 | Baki ₹22,400 this month”  
  (Toggle on/off + choose quiet hours)

#### 4. Family Mode (Newlywed & Joint Family Approved)
- Invite spouse/parent with phone number  
- Three permission levels: View only | Spend only | Full control  
- Auto-split rules: Rent 60:40 | Groceries 50:50 | Kids 100% from dad  
- “Shaadi Gift Vault” – put all wedding cash here → auto-save for home down payment  
- Monthly family report PDF on WhatsApp

#### 5. Tax Engine (Replaces ₹10k–20k CA)
Supports ITR-1 to ITR-4 in one flow:
- Auto pulls Form-16, Form 26AS, AIS  
- Freelance invoice scanner → Section 44ADA or actual expense  
- Rental income → auto 30% standard deduction + TDS match  
- Capital gains → Demat API (Zerodha, Groww) → LTCG/STCG + indexation  
- Agriculture income → exempt but pushes slab if >₹5,000  
- Old vs New regime comparison in 1 screen  
- E-file + e-verify in-app (Aadhaar OTP)  
- ₹999 one-time or free with annual subscription

#### 6. Home Down-Payment Goal (Newlywed Special)
- Auto calculates max affordable EMI (40% of total family income)  
- Shows exact ₹ needed for 20% down payment in chosen city  
- Daily progress bar on home screen  
- PMAY / state subsidy eligibility check + application link  
- “Delay Penalty Calculator” for under-construction projects

#### 7. Subscription & Bill Tracker
- Auto-detects all subscriptions (Netflix, Hotstar, Amazon Prime, gym)  
- 3-day-before reminder + one-tap pay or cancel  
- “Cancel kar do” voice command works

#### 8. Smart Notifications (User Controls Everything)
User sets in 2 minutes:
- Daily brief time (8 PM default)  
- Weekly summary every Sunday 10 AM  
- Low balance alert (when any pocket < ₹2,000)  
- Big expense alert (> ₹3,000 in one go)  
- Tax deadline alerts 30/15/7 days before  

#### 9. Languages (Day 1 Launch)
Hindi, Tamil, Telugu, Bengali, Marathi, Gujarati, Kannada, Malayalam, Punjabi, Odia, Assamese (11 total)

#### 10. Missing Features You Asked by Real Users on X (Added)
- Gold loan comparison in 60 seconds (Rupeek vs Muthoot vs bank)  
- Mutual fund SIP pause/resume in one tap  
- Credit card reward maximiser (which card to use where)  
- “Bhai paisa bhej” voice command → UPI to contact  
- Dark mode + big font mode for parents  

### User Flow – Professional & Fast (No Noob Steps)

1. Open app → Home screen shows pockets + today spend  
2. Scroll → See last 10 transactions (auto categorised)  
3. Tap any transaction → Split | Move to pocket | Add note  
4. Bottom bar: Home | Goals | Tax | Family | Profile  
5. Every night 8 PM → One notification (can snooze forever if want)
